Why do we need kawal C1 if we have KPU and kawal pemilu?

This is a good question. Because of the fantastic kawal pemilu crowd-sourcing initiative, there is no great need for an additional automated verification of the C1 based count.
However, for coming elections this software could improve both the quality and speed of the manual verification.

Quality

The number of C1 forms reported there as problematic is more than 5% for some provinces. Kawal C1 can improve this situation. By detecting problems in the form (unclear handwriting, calculation errors, missing witness signatures, etc.) and giving the uploader the opportunity to type in the totals.

Speed

The manual counting teams can focus their efforts on forms that have been flagged by kawal C1 as problematic. The numbers of forms that are not problematic will immediately be available.

What is the accuracy of the software?

This is something we cannot give reliable numbers on yet. This website should be seen as a proof of concept. However, since all C1 forms and both the official KPU and kawal pemilu resuls are openly available, it will be relatively easy to test and improve the accuracy. This our top priority now.

Does this software only work for pilpres 2014 C1 forms?

At the moment yes. But it will not be difficult to configure it for other forms like the ones that are used for local elections for example.

Who can use this software?

This software is completely open source.

Is it not better to trust humans than to trust a computer?

Yes. This software is only a tool to improve human verification.
